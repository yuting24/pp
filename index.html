Hi Guys!


          面對新興時代來臨，社會大眾獲取資訊的管道已不再受限於傳統報章雜誌或是電視新聞，而隨之而來的是產業結構產生變化，出現了像是youtuber、網紅、電競主播、職業電競選手……等新創職業，我們看中了背後龐大的商機，希望能夠建構一個專門的開箱網站，主要受眾為素人大眾，開箱分享日常所見，希望打著”素人即名人”的口號，成為專業開箱網站，吸引大量使用量。

          PickPack是一個專門給用戶分享各種開箱文及影片，以及彙整其他網站資訊的平台。取名來自於「選」、「包裹」兩個字的英文做結合，打開網頁便能看到琳瑯滿目的開箱文，意味著「你來挑選商品，我來為你開箱」，到平台上能找尋你心目中憧憬的商品，告訴你想知道的一切。

特色功能

素人平台

提供平台給樂於分享的素人，不使網路世界只有名人有機會被看見。

彙整文章/數據分析

1.彙整其他網站貼文連結，讓使用者可多方比較商品
2.分析使用者年齡及點閱類別，評判時下趨勢及族群喜好。

企業徵文

1.快速累積使用者基數:透過徵文累積大量素人使用心得。
2.實驗新品:可向大眾徵求產品貼文，測試市場反應，再改善自己的產品，迎合大眾口味。

圖片搜索

為避免出現使用者不清楚品項名稱卻想搜尋開箱文的情況，我們希望建置圖片搜索，方便使用者透過圖片搜尋。

  • HTML

    屬於基本前端網頁介面框架技術，常搭配CSS、Javascript語言讓網頁多樣化，一起創建互動式網站介面。HTML語言可支援多種方式呈現，例如表格、表單、圖片、文字、連結等等。

  • CSS

    用來調整介面排版及樣式，讓版面有一致性，美化後的頁面讓網頁能更加清楚表達，瀏覽者在使用上簡潔舒服、方便使用。

  • Bootstrap

    一組用於網站和網路應用程式開發的開源前端框架，包括HTML、CSS及JavaScript的框架，旨在使動態網頁和Web應用的開發更加容易。

  • PHP

    PHP為現今常用的網路網頁程式撰寫語言，此語法借鑑了許多常用語言，如:JAVA、C等等，且本身語法簡單，所以容易上手，再加上我們的網站需要利用網路爬蟲技術，來抓取網上的資源，而PHP能實現輕量級的網路爬蟲開發。
